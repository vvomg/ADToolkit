DEPLOYMENT WORKFLOW

SCENARIO 1: First-time deployment
-----------------------------------
$ cd /opt/iva-mail-ansible

# Set passwords via extra vars
$ ansible-playbook playbooks/10-awx-setup.yml \
    -e "awx_admin_password=YourStrongPassword123!@# \
        awx_db_password=YourDBPassword456$%^"

SCENARIO 2: Re-run on existing system (idempotent)
----------------------------------------------------
$ ansible-playbook playbooks/10-awx-setup.yml \
    -e "awx_admin_password=... awx_db_password=..."

# Will skip Docker installation
# Will check and update .env.awx if passwords differ
# Will pull latest images
# Will restart containers if needed

SCENARIO 3: Docker-only installation (reusable)
------------------------------------------------
$ ansible-playbook playbooks/10-awx-setup.yml --tags docker

# Just installs Docker and Docker Compose, doesn't touch AWX

SCENARIO 4: AWX configuration only (if Docker exists)
-------------------------------------------------------
$ ansible-playbook playbooks/10-awx-setup.yml --skip-tags docker

# Skips Docker setup, just deploys AWX containers
