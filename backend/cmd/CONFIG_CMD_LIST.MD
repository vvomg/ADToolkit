# Настройки ИИ агента: команды CMD-интерфейса

Ниже приведён список команд из документа `CMD-methods.md`, сгруппированный по двум критериям:

1. Команды, содержащие в названии подстроку **`config`** (работа с конфигурацией).
2. Команды, выводящие информацию об объектах (доменах, аккаунтах, группах, мейлбоксах, сессиях, правах, справочнике, плагинах и т.д.).

Этот файл может использоваться ИИ-агентом для быстрого поиска нужных команд административного интерфейса.

---

## Команды с «config» в названии

| Команда | Назначение |
|---------|------------|
| `ModuleReadConfig` | Выдаёт словарь с настройками модуля. |
| `ModuleUpdateConfig` | Обновляет настройки модуля. |
| `DOMAINREADCONFIG` | Выдаёт объект настроек домена. |
| `DOMAINUPDATECONFIG` | Обновляет настройки домена. |
| `OBJECTREADCONFIG` | Выдаёт объект настроек объекта (учётной записи, группы и т.п.). |
| `OBJECTUPDATECONFIG` | Обновляет настройки объекта. |

---

## Команды, выводящие информацию об объектах

### Общие (сервер, модули, очереди, логи, нити, соединения)

- `SchemaDump` – описания полей для настроек по схеме.
- `ModulesList` – список инициализированных модулей.
- `MailModulesList` – список модулей обработки почты.
- `MailQueueList` – информация о почтовой очереди модуля.
- `MailBatchList` – информация об ожидающих/активных блоках доставки.
- `MailQueueInfo` – информация о сообщении в очереди по `queueID`.
- `DigestList` – доступные методы хэширования.
- `CipherList` – доступные методы шифрования.
- `KDFList` – доступные методы преобразования паролей в ключи.
- `ECList` – названия доступных эллиптических кривых.
- `ThreadsList` – список активных нитей исполнения.
- `ConnectionsList` – список активных сетевых соединений.
- `LogsList` – список лог-файлов.

### Домены

- `DOMAINSLIST` – список зарегистрированных доменов.
- `DOMAINGETID` – UID домена по имени.
- `DOMAINLISTNAMES` – все имена (основное + псевдонимы) домена.
- `DOMAINGETFLAGS` – флаги применения для имени домена.
- `DOMAINGETSETTING` – значение настройки домена по ключу.
- `DOMAINGETMULTISET` – несколько настроек домена верхнего уровня.
- `DOMAINGETWEBACCESSURL` – URL веб-сервера в домене.
- `KerberosKeyList` – список записей keytab Kerberos для домена.

### Объекты в доменах (учётные записи, группы, ресурсы, переадресаторы)

- `IndexGetEntry` – запись из индекса домена или индекса объектов.
- `OBJECTSLIST` – список объектов в домене.
- `OBJECTGETID` – UID объекта по его имени.
- `OBJECTLISTNAMES` – все имена объекта.
- `OBJECTGETFLAGS` – флаги применения для имени объекта.
- `OBJECTGETSETTING` – значение настройки объекта по ключу.
- `ObjectGetMultiset` – несколько настроек объекта верхнего уровня.

### Аккаунты (учётные записи)

- `AccountListNamespaces` – пространства имён почтовых папок аккаунта.
- `AccountListMailboxes` – список почтовых папок аккаунта.
- `AccountGetMailStorageSize` – размер почты (сумма размеров мейлбоксов) в аккаунте.
- `DataFilesList` – информация о файлах или папках аккаунта по пути.
- `ObjectGetInfo` – значение метаданных объекта по ключу.
- `AccountGetRights` – административные права, выданные аккаунту.

### Ресурсные аккаунты

- `ResourceGetOwner` – имя владельца ресурсного аккаунта.

### Группы

- `GroupListMembers` – список членов группы (или список групп для аккаунта).
- `GroupHasAccount` – проверка наличия члена в группе.
- `GroupGetOwner` – имя владельца группы.

### Мейлбоксы и сообщения

- `MAILBOXESLIST` – все мейлбоксы аккаунта.
- `MboxMessageInfo` – информация о сообщении в мейлбоксе.
- `MailboxGetACL` – ACL (список управления правами) мейлбокса.
- `MailboxGetMyRights` – права аккаунта на мейлбокс.
- `CalendarGetACL` – ACL календаря.
- `CalendarGetMyRights` – права аккаунта на календарь.
- `MboxGetInfo` – метаданные почтовой папки по ключу.
- `MessageGetAttribute` – значение атрибута сообщения.

### Административные права

- `RightsList` – список административных прав, доступных запрашивающему администратору.
- `AccessDump` – список зарегистрированных ролей и администраторов.
- `RolesList` – список административных ролей.
- `RoleGet` – список прав для роли.
- `AdminsList` – список административных аккаунтов.
- `AdminInfo` – права и роли для администратора.

### Справочник (Directory / LDAP / PostgreSQL)

- `DirectoryFind` – поиск записей в справочнике.
- `PgDirectoryFind` – поиск в справочнике, хранимом в PostgreSQL.
- `LdapClientSearch` – поиск через LDAP-клиент.

### Сессии

- `SESSIONLIST` – список сессий (фильтрация по домену/аккаунту).
- `SESSIONGETINFO` – данные сессии по токену.

### Правила обработки почты

- `RulesListOptions` – список операций, условий и действий.
- `RulesList` – список правил на уровне сервера, домена или объекта.
- `RuleGet` – конфигурация правила по имени.

### Плагины

- `PluginList` – список плагинов с настройками и статусом.

### Хранилище некорректных файлов

- `DefectedFileList` – список файлов в хранилище.
- `DefectedFileGet` – содержимое указанного файла в base64.

### Белые и чёрные списки адресов

- `GetWhitelisted` – белый список сетевых адресов.
- `GetBlacklisted` – чёрный список сетевых адресов.

### Календари

- `CalendarSearch` – записи календаря за заданный интервал времени.
- `CalendarShow` – информация о календарном событии по UID или ID сообщения.
- `CalendarNextAlarm` – ближайшее напоминание.
- `CalendarFreeBusy` – времена занятости.
- `TimeZoneList` – список известных серверу временных зон.

### Контакты

- `ContactsSearch` – поиск контактов по параметрам.

### Полнотекстовый поиск

- `FullTextSearch` – поиск по полнотекстовому индексу почтовой папки.

### Профили SMTP-Input и SMTP-Output

- `smtpiProfilesList` – список профилей SMTP-Input.
- `SMTPOProfilesList` – список профилей SMTP-Output.

### Списки адресов (Address Lists)

- `AddressListCheck` – проверка наличия адреса в списке.
- `AddressListGet` – получение именованного списка адресов.

### Федеративный доступ

- `FederatedSharingList` – список единиц настроек федеративного доступа.

### Планировщик задач

- `TasksList` – список запланированных заданий.

### Отложенные почтовые отправления

- `DeferredDeliveryList` – список отложенных отправлений (макс. 100).

### Статические переменные

- `GetStaticVar` – объект, хранимый в статической переменной по OID.
